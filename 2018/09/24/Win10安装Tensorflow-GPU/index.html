Win10安装Tensorflow(GPU)

发表于 2018-09-24 | 更新于 2018-10-04 | 分类于 学习 | 阅读次数：

前段时间在win 10上安装了Tensorflow，在这里记录下安装过程和配置情况。

安装Anaconda

在网上下载Anaconda3的安装包，右键Anaconda3-5.2.0-Windows-x86_64，选择使用管理员模式运行，选择安装到当前计算机的所有用户上，安装python3.6以及加入PATH

在安装完成后可以在Anaconda Prompt中输入命令conda update conda更新conda，输入命令conda upgrade --all来更新所有包

Tensorflow 1.10安装

打开Anaconda Prompt，创建一个名为tensorflow的环境，命令如下conda create -n tensorflow python=3.6

开启环境activate tensorflow

输入命令pip install --ignore-installed --upgrade tensorflow-gpu进行安装

验证：

若输出b'Hello, TensorFlow!'即为成功

检测tensorflow的确用gpu来做运算

在输出中device:GPU:0的意思就是说该运算用到了GPU
