Hexo + Github搭建博客

发表于 2018-08-17 | 分类于 Hexo | 阅读次数：

把自己使用Hexo + GitHub搭建博客过程分享一下

环境和工具准备

Git

打开Git安装程序，选择要安装的组件，可以全选也可以默认，然后一路next即可，安装路径根据自己习惯更改。

打开cmd命令窗口，执行git --version命令，结果显示Git版本，安装成功。

安装Node.js

同样打开Node.js安装程序，一路默认即可，安装路径根据自己习惯更改。

打开cmd命令窗口，执行node -v命令，显示Node.js版本，安装成功。

链接到Github

新建一个项目githubname.github.io

配置SSH key

打开Git Bash

生成密匙

生成的密匙存放位置提示里有。

把SSH key添加到Github

复制刚刚的密匙文件（后缀为pub）的内容，打开github右上角的个人设置，有SSH and GPG keys一项，新建一个填入密匙即可。

检查

打开Git Bash

如果得到以下反馈

输入yes回车

没密码直接回车就行，然后就ok

部署到Github

安装hexo-deployer-git

网站配置

修改 hexo 文件夹下的 _config.yml 全局配置文件，修改 deploy 属性代码，将本地 hexo 项目托管到 GitHub 上

详细参考官方文档Hexo

个性化

配置博客个人信息

打开Hexo的全局配置文件_config.yml，可以添加站点名、副标题、信息简介等等
